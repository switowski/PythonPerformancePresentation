Let's start with something simple. Let's say you want to count the number of elements in a list. You can easily write you own function with just a for loop that increments a variable. There is nothing wrong with this code, except that it's slow. You can achieve the same result using the built-in len() function. Any difference ? Yes, the second function is way faster. So, try to use the built-in functions when you can. There is just a few of them, so it's very easy to remember the most useful ones.
